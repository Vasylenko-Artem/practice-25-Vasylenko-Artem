# Обчислювальна практика

## Опис проекту
Цей проект містить лабораторні роботи, виконані під час літньої обчислювальної практики. Практика охоплювала різні напрямки програмування на C++ та Java, включаючи обробку виключень, багатопоточне програмування, шаблони проектування, роботу з базами даних та графічне відображення даних.

Проект демонструє поєднання алгоритмічних задач, об’єктно-орієнтованого підходу та практичних навичок роботи з GUI і базами даних.
